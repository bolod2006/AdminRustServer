# 📋 Changelog — RustAdminServer

## [0.2.0] — 2024-XX-XX

### ✨ Added
- Modern Discord-style navigation tabs with colored icons
- Animated sliding tab indicator with per-tab color sync
- Redesigned Settings menu — dark theme, shadows, hover effects
- RustMaps API integration for server map rendering
- Steam API Key configuration for VAC checks
- Companion Oxide plugin for building/cupboard tracking (Pro)
- Search bar for Offline and Bans tabs
- Country filter dropdown
- Hardware-bound Pro license system

### 🔧 Changed
- Navigation bar height: 22px → 40px
- Tab icons redesigned with colored backgrounds
- Menu popup updated to dark rounded theme

### 🐛 Fixed
- DataGrid selection style on dark background
- Window drag on multi-monitor setups
- ComboBox dropdown style inconsistency

## [0.1.0] — 2024-XX-XX

### ✨ Added
- Initial public release
- RCON WebSocket connection
- Player management: Online, Offline, Bans
- Server console with colored output
- Chat monitor with send functionality
- Live map with real-time player positions
- GeoIP country detection with flag icons

- Admin actions: Ban, Kick, Kill, Mute, Sleep
- Copy player info: SteamID, IP, Name

- Scheduled commands (Pro)
